Defaults mirror HeroUI [constants.ts](https://github.com/heroui-inc/heroui/blob/v3/packages/react/src/components/toast/constants.ts) (`DEFAULT_TOAST_TIMEOUT` 4000 ms, `DEFAULT_GAP` 12, `DEFAULT_MAX_VISIBLE_TOAST` 3, `DEFAULT_TOAST_WIDTH` 460).
